Having trouble?

Troubleshooting

  • Time Synchronization: Ensure the time on your phone is synchronized with internet time. Mismatched time can cause verification code errors.
  • Authenticator App Issues: If you're having trouble with one authenticator app, try using another one like Google Authenticator or Authy.
  • QR Code not Scanning: If the QR code is not scanning, try increasing your screen brightness or using the manual entry code.
